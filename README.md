# @amlplugins/salesforce-marketing-cloud

Native `salesforce-marketing-cloud-sdk` SDK re-exported under the `@amlplugins` namespace.

> Salesforce Marketing Cloud — Email Studio, Mobile Studio, Journey Builder, Audience Builder REST.

## Install

```bash
npm install @amlplugins/salesforce-marketing-cloud
```

## Use

```ts
import { /* named exports from salesforce-marketing-cloud-sdk */ } from "@amlplugins/salesforce-marketing-cloud";
// or
import sdk from "@amlplugins/salesforce-marketing-cloud";
```

Built and published by GitHub Actions on every push to `main`.

Part of the amlplugins family — see https://github.com/amlplugins.
